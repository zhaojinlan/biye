## 第一阶段：补底线（1-2个月，最高优先级）

> 这些是面试必考项，缺了会被一票否决

### 1. Transformer原理 🔲

**为什么必学**：面试官问"讲一下Attention机制"你答不上来，后面的项目经历都白聊。

| 学习内容 | 推荐资源 | 预计时间 |
|----------|----------|----------|
| Self-Attention机制（Q/K/V计算） | Jay Alammar《The Illustrated Transformer》 | 2天 |
| Multi-Head Attention | 同上 + 李沐《动手学深度学习》对应章节 | 1天 |
| 位置编码（Sinusoidal → RoPE） | 苏剑林博客《Transformer升级之路》 | 2天 |
| Layer Norm vs Batch Norm | 任意深度学习教材 | 半天 |
| KV-Cache推理优化 | 知乎搜"KV Cache原理详解" | 1天 |
| 手写一遍Self-Attention（PyTorch） | 自己动手，不超过50行代码 | 1天 |

**验收标准**：能在白板上画出Transformer架构图，手推Attention计算过程，解释为什么需要除以√d_k。

### 2. 模型微调实战 🔲

**为什么必学**：区分"调API的"和"做大模型的"的分界线。做完一个就能写进简历。

| 学习内容 | 推荐资源 | 预计时间 |
|----------|----------|----------|
| LoRA/QLoRA原理 | 原论文 + 知乎解读（搜"LoRA原理详解"） | 1天 |
| LLaMA-Factory安装与使用 | GitHub: hiyouga/LLaMA-Factory，README非常详细 | 1天 |
| **实战：用LoRA微调Qwen做医疗问答** | 用你知识图谱中的数据构造SFT数据集，LLaMA-Factory微调 | 3-5天 |
| 微调数据集构造（Alpaca/ShareGPT格式） | LLaMA-Factory文档中的数据格式说明 | 1天 |
| 训练参数调优（lr/epoch/rank/alpha） | 实践中学习，先用默认参数跑通再调 | 2天 |

**验收标准**：能说"我用LLaMA-Factory对Qwen2.5-7B做了LoRA微调，数据集是从知识图谱中构造的X条医疗问答，训练了Y个epoch，在测试集上的表现是Z"。

## 推荐学习资源汇总

### 必读博客/文章
1. **Jay Alammar** — The Illustrated Transformer（图解Transformer，入门必读）
2. **Lilian Weng** — LLM Powered Autonomous Agents（Agent综述，写得极好）
3. **苏剑林（科学空间）** — Transformer升级之路系列（中文最好的Transformer深度解析）
4. **LangChain官方博客** — 紧跟RAG/Agent最新实践

### 必看视频
1. **李沐** — 《动手学深度学习》Transformer章节（B站）
2. **Andrej Karpathy** — Let's build GPT from scratch（YouTube，手把手实现GPT）
3. **吴恩达** — LangChain/LangGraph短课程（DeepLearning.AI，免费）

### 必跟GitHub仓库
1. `langchain-ai/langgraph` — 你的主力框架，紧跟更新
2. `hiyouga/LLaMA-Factory` — 微调必备工具
3. `vllm-project/vllm` — 推理部署标杆
4. `explodinggradients/ragas` — RAG评测框架
5. `astral-sh/ruff` — Python代码质量工具

### 面试八股速查
- 知乎专栏搜"大模型面试题汇总"
- GitHub搜"LLM-interview"（有多个高星仓库）
- 重点准备：Transformer原理、Attention计算、位置编码、KV-Cache、LoRA原理、RAG流程、Agent设计模式
