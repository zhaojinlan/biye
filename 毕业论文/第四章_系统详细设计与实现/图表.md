## 图 4-1 知识图谱构建详细流程（含数据流）

```mermaid
graph TD
    PDF[PDF医学文献] --> DOCLING[Docling解析器<br/>输出HTML]
    DOCLING --> CLEAN[BeautifulSoup清洗<br/>去DOI/参考文献/噪声标签]
    CLEAN --> MD[转换Markdown<br/>保留标题层级结构]
    MD --> CHUNK[RecursiveCharacterTextSplitter<br/>chunk_size=2000<br/>chunk_overlap=200]
    CHUNK --> LLM_EXT[LLM实体关系抽取<br/>Qwen2.5-14B<br/>结构化JSON输出]
    LLM_EXT --> ENT["输出实体列表<br/>Disease/Symptom/RiskFactor<br/>Pathogen/Treatment/DiagnosticTest"]
    LLM_EXT --> REL["输出关系列表<br/>HAS_SYMPTOM/TREATED_BY<br/>DIAGNOSED_BY/HAS_RISK_FACTOR/CAUSED_BY"]
    ENT --> NEO4J[Neo4j写入<br/>MERGE去重]
    REL --> NEO4J
    ENT --> VEC[M3E-Base向量化<br/>症状节点768维嵌入]
    VEC --> REDIS[Redis向量索引<br/>余弦相似度检索]
```

## 图 4-4 风险评估 Softmax 概率计算流程

```mermaid
graph TD
    IN[患者症状关键词列表] --> QUERY["Neo4j向量检索<br/>每个症状 → Top-K匹配疾病"]
    QUERY --> AGG["聚合疾病匹配度<br/>每个疾病的匹配症状数"]
    AGG --> RF["风险因子加权<br/>匹配风险因子 → 额外加分"]
    RF --> SCORE["原始得分向量<br/>S = s1, s2, ..., sn"]
    SCORE --> SOFTMAX["Softmax归一化<br/>P_i = exp(s_i) / Σexp(s_j)"]
    SOFTMAX --> TOPN["取Top-N疾病<br/>按概率降序排列"]
    TOPN --> MCP["MCP工具调用<br/>get_diagnostic_tests_for_disease"]
    MCP --> OUT["输出风险评估报告<br/>疾病名称 + 概率 + 推荐检查"]
```
